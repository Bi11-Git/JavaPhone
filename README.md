
# javaPhone

To javaPhone είναι ένα πρόγραμμα διαχείρησης συμβολαίων τηλεπικοινωνιών. Παρέχει λειτουργίες όπως εισαγωγή προφίλ πελάτη,
εισαγωγή συμβολαίου, διαγραφή συμβολαίου και εμφάνιση όλων των ενεργών συμβολαίων και στατιστικών.

## Usage

Compile using 

```
mvn compile
```

Create a jar using 

```
mvn package
```

Run main using 

```
java -cp target/it219118-1.0-SNAPSHOT.jar org.hua.it219118.IT219118
```
